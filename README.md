📊 Amazon Prime Content Analysis Dashboard
📌 Project Overview
This project is an interactive Power BI dashboard built using the Amazon Prime Titles dataset. The dashboard analyzes content distribution across genres, ratings, countries, release years, and content types to uncover useful insights and trends.
The goal of this project was to practice:

Data visualization
Dashboard design
Data analysis
Power BI fundamentals
Basic DAX measures


🛠 Tools & Technologies Used
Microsoft Power BI
Power Query
Basic DAX
Data Cleaning & Transformation



📂 Dataset Information
The dataset contains information about Amazon Prime content, including:

Title
Genre
Country
Rating
Release Year
Director
Type (Movie / TV Show)


📈 Dashboard Features
🔹 KPI Cards

Total Titles
Total Ratings
Total Genres
Total Directors
Start Year
End Year


🔹 Visualizations

Genre Distribution Analysis
Rating Distribution
Movies vs TV Shows Comparison
Country-wise Content Distribution (Map)
Release Year Trend Analysis



🎯 Key Insights


Movies dominate the platform compared to TV Shows
Drama is the most common genre
Content production increased rapidly after 2000
Majority of titles are concentrated in a few countries
TV-MA and 13+ are among the most common content ratings
